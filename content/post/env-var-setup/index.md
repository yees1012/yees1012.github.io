# 配置环境变量

1. settings 搜索 variable，选择 `Edit the system environment variables`
   
   ![alt text](image-8.png)

2. 点击 `Environment Variables`
   
   ![alt text](image-9.png)

3. 点击 `New` 添加环境变量，然后点击 `OK` ， `Apply`

    ![alt text](image-10.png)
